# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2025-01-XX

### Добавлено
- Авторизация по телефону/паролю и OTP
- Dashboard с KPI карточками
- Список заявок с фильтрами и поиском
- Детальный просмотр заявок с таймлайном статусов
- Создание заявок через многошаговый мастер
- Каталог техники, услуг и материалов
- Финансовые отчёты
- Профиль пользователя с настройками
- Push-уведомления с deep links
- Оффлайн режим с кэшированием и очередью синхронизации
- Настройки уведомлений с синхронизацией с backend

### Изменено
- Обновлена архитектура на feature-based модули
- Улучшена обработка ошибок

### Исправлено
- Исправлены проблемы с refresh токенами
- Улучшена работа оффлайн режима
