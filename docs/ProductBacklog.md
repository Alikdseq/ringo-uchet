## Product Backlog — роли и эпики

### 1. Админ (владелец)
- **Управление пользователями и ролями**
  - CRUD пользователей, назначение ролей, блокировка/разблокировка.
  - Настройка тарифов, налогов, штрафов, SLA.
- **Глобальные настройки**
  - Параметры pricing engine, шаблоны PDF, доступ к интеграциям (S3, почта, SMS, платежи).
- **Дашборды и отчёты**
  - KPI за период, загрузка парка, топ-5 машин, сравнение выручки/расходов.
  - Фильтры по дате/технике/локации, экспорт CSV/XLSX/PDF.
- **Безопасность и аудит**
  - Просмотр audit log, настройка уведомлений о критичных событиях.

### 2. Менеджер
- **Заявки**
  - Создание/редактирование заявок, выбор клиента, адреса (карта), сроков, техники/услуг/материалов.
  - Назначение оператора, согласование с клиентом, изменение статусов до `APPROVED`.
- **Расчёт стоимости**
  - Просмотр предварительного расчёта, ручные корректировки (скидки, доп. услуги).
- **Вложения и коммуникации**
  - Загрузка документов, фото, комментарии, отправка счёта клиенту.
- **Отчёты по менеджеру**
  - Личная выручка, количество заявок, статус согласований.

### 3. Оператор / Сотрудник
- **Назначения**
  - Список своих рейсов, push-уведомления о новых задачах.
  - Подтверждение выезда, отметка начала/конца работ.
- **Фотопротокол**
  - Фото до/после с GPS, добавление расходников.
- **Рабочие отчёты**
  - Заполнение фактического времени, материалов, комментариев.
- **Профиль**
  - Просмотр заработка, статуса выплат, настройки уведомлений.

### 4. Бухгалтер / Финансист
- **Финансы**
  - Просмотр/подтверждение счетов, акты, платежи.
  - Учёт расходов (топливо, ремонт), загрузка чеков.
- **Зарплаты**
  - Расчёт SalaryRecord по заказам, утверждение выплат.
- **Отчёты**
  - Финансовая отчётность (доходы/расходы/маржа), детализация по технике и сотрудникам.
- **Экспорты**
  - CSV/XLSX для бухгалтерских систем, интеграция с банком (опционально).

### 5. Клиент (публичный фронт, опционально)
- **Форма заявки**
  - Запрос техники с описанием работ, контактами.
- **Статус заказа**
  - Проверка статуса, получение PDF счёта/акта.
- **Коммуникация**
  - Обратная связь, подтверждение дат/стоимости.

---

Каждый эпик дробится в таск-трекере на пользовательские истории и атомарные задачи согласно `AfinaEcosystemPlan.md`.
