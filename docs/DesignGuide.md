### Типографика

- Заголовки: `SF Pro Display` / `Inter` (веб), `SF Pro / Roboto` (мобильное).
- Основной текст: `Inter 16/24` (веб), `SF Pro Text 16/24` (mobile).
- Мелкий текст/лейблы: 14/20.

### Компоненты UI (пересечение Flutter + Next.js)

- **AppBar/Header**: логотип, переключатель языка, иконки уведомлений, профиль.
- **Cards**: тень 2dp, радиус 12px, padding 16.
- **Buttons**:
  - Primary (solid, primary color, радиус 10, высота 48).
  - Secondary (outline).
  - Status actions — full width с цветом статуса.
- **Badges**: pill с цветом статуса, текст белый, uppercase.
- **Forms**: floating labels, иконки статуса (ошибка/успех), валидаторы в реальном времени.
- **Charts**: fl_chart (mobile) / ApexCharts (web), единый цветовой набор.

### UX паттерны

- Status change → модалка с комментарием + опциональное фото/подпись.
- Фото просмотр → lightbox/fullscreen, pinch-to-zoom, подписи «До/После».
- Offline indicator → баннер сверху + иконка в AppBar.
- Карты → Google Maps SDK (Flutter) / Mapbox/Google (web).
- PDF actions → отдельное меню (скачать, поделиться, отправить email/WhatsApp).

### Адаптивность

- Веб: брейкпоинты 320/768/1024/1440. Публичный сайт — mobile-first.
- Мобильное: поддержка iOS 13+/Android 8+, landscape lock на операционных экранах (по согласованию).

Гайд расширяется дизайнером: компоненты Figma, токены цвета/типа, иконочный набор, micro-animation правила.
