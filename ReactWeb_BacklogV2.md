## Backlog v2 для React Web фронта Ringo Uchet

Список приоритетных улучшений после первой стабильной версии.

---

## 1. Улучшения UX

- Единый Date/DateTime‑picker с поддержкой быстрых пресетов («сегодня», «7 дней», «месяц»).
- Улучшение мастера создания заявки:
  - визуальный прогресс‑бар по шагам;
  - inline‑валидация полей (маска телефона, подсказки по адресу);
  - сохранение черновика заявки локально.
- Более плотная компоновка таблиц на десктопе и упрощённый список на мобайле.
- Подсветка критичных статусов (просроченные заявки, заявки без оператора и т.п.).

---

## 2. Новые отчёты

- Расширенные фильтры в `/reports/*`:
  - по оператору/технике/клиенту;
  - по типам работ/категориям услуг.
- Графики загрузки техники по календарю (heatmap/календарь смен).
- Экспорт отчётов в XLSX/PDF (совместно с backend).

---

## 3. Расширенный оффлайн

- Поддержка оффлайн‑режима для:
  - редактирования существующих заявок;
  - добавления фото/приложений с последующей синхронизацией.
- UI‑индикация состояния сети и оффлайн‑очереди:
  - бейдж в навигации «Оффлайн очередь» с количеством элементов;
  - статус‑бар при потере/восстановлении сети.

---

## 4. Интеграции с платёжными шлюзами

- Подготовить фронтовую интеграцию для приёма онлайн‑оплаты:
  - карта/СБП (конкретный провайдер зависит от выбранного backend‑решения).
- Отображение статуса оплаты в заявках и отчётах:
  - пометка оплаченных заявок;
  - сверка сумм по отчётам.
