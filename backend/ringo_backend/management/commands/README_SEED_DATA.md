# Команда seed_data

## Описание

Команда `seed_data` добавляет начальные данные в базу данных:
- Технику с ценами (почасовая оплата 2450₽, дневные ставки из списка)
- Услуги (19 услуг в 11 категориях)
- Грунт (8 видов)
- Инструменты (8 видов)
- Операторов (2 пользователя)

## Использование

```bash
# Через Docker
docker compose exec django-api python manage.py seed_data

# Или локально (если Django настроен локально)
python manage.py seed_data
```

## Безопасность

Команда использует `get_or_create`, поэтому её можно запускать многократно без дублирования данных. Если запись уже существует, она будет обновлена.
